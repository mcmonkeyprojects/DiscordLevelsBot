Discord Levels Bot



Discord Levels Bot is a bot for Discord that provides automatic level-ups based on message activity.


The bot is fully open source, view the source code and learn more at the DiscordLevelsBot GitHub page.

If you're lost... you're probably not meant to be here. If you're looking for the leaderboard in a Discord guild, try /leaderboard inside that guild.
